# Подготовка к демоэкзамену

**Подготовка к демонстрационному экзамену будет происходить по следующим модулям:**

## Модуль 1: Анализ и проектирование требований, бизнеспроцессов

UML-Diagramm
## Модуль 2: Проектирование реляционного хранилища данных

ER-Diagramm, DataBase.
## Модуль 3: Разработка баз данных, объектов баз данных и импорт

Import from Execel, csv, txt, cod.
## Модуль 4: Разработка desktop-приложений

DesktopApp - созданное с учётом требований и пожеланий, исходя из ТЗ

## Модуль 5: Разработка приложений для мобильных платформ

Mobile Apps - разработка мобильного приложения используя Android Studio
## Модуль 6: Создание инсталляторов

Создание инсталлятора, с помощью которого будет происходить демонстрация готового продукта
## Модуль 7: Тестирование программных решений

Unit-test, библиотека классов, проверка согласно ТЗ
## Модуль 8: Документирование программных решений

word-шаблон, проверка умения работы с Word

## Модуль 9: Общий профессионализм решения

Оценка продукта согласно ТЗ, проверка того, как разрабативалась программа, что было использовано, как и так далее.
